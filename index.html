Unix Cheat Sheet

[ECHAP] pour effacer
Commande Détails
cd Se déplacer dans un dossier
cd ~ Se déplacer dans le dossier de l'utilisateur courant
cd - Retourner au dossier précédent
ls -la Afficher le contenu du dossier courant, avec les fichier cachés
mkdir Créer un dossier
rmdir Supprimer un dossier vide
rm -rf Forcer la suppression d'un dossier et son contenu
cp -r Copier un dossier et son contenu
cat Afficher le contenu d'un fichier
more Afficher le contenu d'un fichier page par page (Espace/Entrée pour naviguer, Q pour quitter)
head Afficher les 10 premières lignes du fichier
tail Afficher les 10 dernières lignes du fichier
tail -f Afficher les 10 dernières lignes du fichier en temps réel (Ctrl+C pour quitter)
touch Créer un fichier
rm Supprimer un fichier
rm -f Forcer la suppression d'un fichier
cp Copier un fichier
nano Éditer un fichier avec nano
vi Éditer un fichier avec vi
